마작 기록용 모바일 홈페이지 기획


1. 모바일 기준
2. 4인 마작 기준

## 페이지 구성 ##

1. 대국 기록
  1-1. 게임 설정  (온라인 1/3배, 오프라인 1배)
  1-2. 플레이어 설정  (대국에 누가 참여했냐, 오야 설정, 동서남북 설정)
  1-3. 각 대국은 종료시 데이터만 기록한다.

2. 대국 내역 조회

3. 랭킹 (1개 페이지, 필터로 분류)
  3-1. 필터: 월별 / 분기별 / 통산
  3-2. 순위 히스토리 (기간별 순위 변화 이력)
  3-3. 순위 변동 (전 기간 대비 순위 오르내림 표시)

4. 회원 목록
  4-1. 플레이어 추가/수정/삭제

5. 통계


---

## 페이지 상세 설계 ##

---

### 1. 대국 기록

단계별 흐름 (Step 1 → 2 → 3)

**Step 1. 게임 설정**
- 온라인 / 오프라인 선택 (토글 또는 라디오 버튼)
- 온라인 선택 시: 배율 선택 (1배 / 3배)
- 오프라인은 1배 고정 (배율 선택 UI 숨김)
- 다음 버튼 → Step 2로 이동

**Step 2. 플레이어 설정**
- 등록된 회원 목록에서 4명 선택
- 오야(기준 플레이어) 지정
- 동/서/남/북 방위 배치 (4명에게 각각 지정)
- 다음 버튼 → Step 3으로 이동

**Step 3. 결과 입력**
- 4명 각각의 최종 점수 입력
- 점수 기준으로 등수 자동 계산 (동점 처리 고려)
- 저장 버튼 → 대국 기록 완료
- 저장 후 대국 내역 조회 페이지로 이동

---

### 2. 대국 내역 조회

**목록 화면**
- 날짜 역순으로 대국 목록 표시
- 각 항목: 날짜, 참여 플레이어 이름, 온/오프라인 구분
- 필터: 날짜 범위, 특정 플레이어 포함 여부
- 항목 클릭 → 상세 화면

**상세 화면**
- 해당 대국의 게임 설정 (온/오프라인, 배율)
- 플레이어별 방위, 점수, 등수
- 수정 / 삭제 버튼

---

### 3. 랭킹

**상단 필터**
- 월별 / 분기별 / 통산 탭 또는 드롭다운
- 월별: 년/월 선택
- 분기별: 년/분기 선택
- 통산: 전체 기간

**랭킹 테이블**
- 순위, 플레이어 이름, 총 점수, 대국 수, 평균 점수, 1위 횟수
- 순위 변동 아이콘 (▲ 상승 / ▼ 하락 / — 유지) — 전 기간 대비

**순위 히스토리 섹션**
- 플레이어 선택 시 해당 플레이어의 기간별 순위 변화 그래프 또는 테이블
- 전체 보기: 모든 플레이어의 순위 흐름을 꺾은선 차트로 표시

---

### 4. 회원 목록

**목록 화면**
- 등록된 플레이어 카드 또는 리스트
- 각 항목: 이름, 누적 대국 수, 간단한 승률
- 우측 상단 + 버튼으로 신규 추가

**추가 / 수정 화면**
- 이름 입력
- (선택) 프로필 이미지 또는 아이콘 색상 지정
- 저장 / 취소

**삭제**
- 삭제 시 확인 다이얼로그 표시
- 대국 기록이 있는 플레이어는 삭제 불가 또는 경고 처리

---

### 5. 통계

**상단 필터**
- 기간 선택 (전체 / 월별 / 분기별)
- 플레이어 선택 (전체 또는 개인)

**전체 통계**
- 총 대국 수, 참여 플레이어 수
- 온/오프라인 비율

**개인 통계 (플레이어 선택 시)**
- 총 대국 수, 1위/2위/3위/4위 횟수 및 비율
- 평균 최종 점수
- 최근 N회 성적 추이 (꺾은선 차트)
- 온라인 / 오프라인 별도 집계
